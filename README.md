# 项目运行
- 注意：该项目中使用ES6/ES7特性、及react-hooks，请保持 node版本在8.0.0及以上。react/react-dom版本在16.8.0及以上。
```bash

项目启动
yarn (推荐) 或 npm i
yarn mock (mock环境)
yarn start (本地开发环境)

项目打包
yarn build-test (测试环境)
yarn build-prod （正式环境）

```
